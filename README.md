# 🚦 Elevator & Traffic Controller (Verilog HDL)

A collection of Verilog HDL projects implementing real-world digital systems using Finite State Machines (FSMs). Each project is fully synthesizable, verified using testbenches, and simulated in Xilinx Vivado.

# 📁 Projects

## 🛗 Elevator Controller

A Verilog FSM-based elevator controller supporting multiple floor requests, directional movement, and door control.

### Features

- FSM Based Design
- Multi-floor Control
- Up & Down Movement
- Door Open/Close Logic
- Testbench Verification
- Synthesizable RTL

➡️ Open the **Elevator_Controller** folder for detailed documentation.

---

## 🚦 Traffic Light Controller

A finite state machine implementation of a traffic signal controller that sequences Red, Yellow, and Green lights.

### Features

- FSM Based Controller
- Clock Driven Operation
- Safe Traffic Signal Sequencing
- Testbench Verification
- Synthesizable RTL

➡️ Open the **Traffic_Light** folder for project details.
